引言

Generating Function 简介

OGF 方法最早是由 Abraham de Moivre 发表于 1730 年。OGF 方法的优点在于它可以把数列问题转换为函数问题，为数列问题的求解引入了一个更广阔的发挥空间。在实际应用中往往会收到简单易编码、强大易维护的效果。

可知 {a1, a2} 的 C 值的 OGF 即是 {a1}, {a2} 两个 C 值的 OGF 的乘积。同理，{a1, a2, a3}的 C 值的 OGF 即是 {a1}, {a2}, {a3} 三个 C 值的 OGF 的乘积，以此可推广得到 {a1, … ak} 的 OGF 为 (1 + x)^k，与采用二项式定理的解法获得的结果是一样的，这里我们所发现的这个有意思的规律正是 Convolution 定理，简述如下：
设 A，B 为相互无重复元素的两个集合；
按规则 r 从 A 中取元素，产生所有结果种类数的 OGF 为 F1；
按规则 r 从 B 中取元素，产生所有结果种类数的 OGF 为 F2；
按规则 r 从 {A, B} 中取元素，产生所有结果种类数的 OGF 为 F3；
则，F3 = F1 * F2；
Convolution 定理正是 OGF 相乘运算的具体实际意义的展现之一。
有了 Convolution 定理之后让我们来解一下上个问题的一个升级版本：如果每次取的元素可以重复，求 C 值。
